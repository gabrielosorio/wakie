# Wakie

## Overview

An RTC-based alarm clock built for no good reason.

### BOM
1. Arduino Nano
1. DS1307 (Tiny RTC)
1. LCM1602C LCD
1. Piezo speaker
1. Momentary switch
1. 1K Resistor
1. 220R Resistor
